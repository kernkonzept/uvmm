# L4Re Uvmm

uvmm is a virtual machine monitor for the L4Re operating system. It
allows to configure and execute guest OSes on top of L4Re, mainly using
Virtio.

# Documentation

This package is part of the L4Re Operating System Framework. For
documentation and build instructions please refer to
[l4re.org](https://l4re.org).
